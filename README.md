# AI-Native Java Backend Bootcamp

> 面向 AI 应用开发者的 Java 后端训练营仓库

## 课程定位

这套训练营不是教“Java 如何调用一个模型接口”，而是教“如何用 Java / Spring Boot 构建可上线的 AI-Native 后端系统”。

- 主线受众：AI 应用开发者
- 兼顾受众：希望补足后端能力的前端工程师
- 主技术栈：Java 21 + Spring Boot 3.x
- 主业务场景：企业知识助手

## 课程主线

整套课程围绕一个可持续演进的企业知识助手展开，逐步实现：

1. 模型接入
2. 对话接口与流式输出
3. Prompt、上下文与结构化结果
4. 文件上传、解析、切片与知识入库
5. RAG 检索增强
6. Redis、Elasticsearch、向量检索协同
7. Tool Calling 与 Agent
8. Workflow 与异步任务
9. AI 工程化治理
10. 业务融合与生产化
